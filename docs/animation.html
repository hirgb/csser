Animation

Usage

"*" Replaced with animation name

“*”替换为动画名称

以动画 fade 为例

  1. 在要使用动画的元素上添加 fade 类，元素自动应用 enter 动画。
  2. 使用触发器：在元素上添加 data-trigger="hover" ，则元素会在 :hover 状态下触发 leave 动画。所有动画都支持 hover, active, focus 三种触发方式。
  3. 使用编程控制：将元素 data-action 属性设置为 enter 或 leave 则会触发相应动画。
